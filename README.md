## Installation

### Ubuntu 

```bash
$ sudo apt install libpcl-dev
$ cd ~
$ git clone https://github.com/udacity/SFND_Lidar_Obstacle_Detection.git
$ cd SFND_Lidar_Obstacle_Detection
$ mkdir build && cd build
$ cmake ..
$ make
$ ./environment
```
